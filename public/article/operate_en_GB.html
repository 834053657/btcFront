Guideline

How to post ads for buying and selling Bitcoin?

Find the publishing ad entrance and choose to publish or sell ads based on your needs.

You can create up to 4 deal ads. When you create an ad, please create the number of ads that suit your needs.


Requirements of ads

You need to set the location, currency, payment method, transaction price, transaction limit, write down the terms of the transaction, and the payment period.

You must have at least 0.05 BTC in wallet of utomarket in order to create and publicly display your ad.

Up to 2 ads can be created in the same country in the same currency.


Options settings of publish ads:

Location

Enter the country you want the ad to show.


Payment methods

Select the payment method you want to provide in the drop-down menu. Learn more about the payment method section.


Currency

CNY/USD

Select the currency in which you want to sell Bitcoin. For example, if you want to sell in China, you should choose CNY RMB. You can use this list to find currency abbreviations.


Market price ratio

The market price mentioned here is a kind of price that we can reflect on the market price based on the real-time price equilibrium of major famous exchanges.


Trading price:

Be free to set your own ideal price.


Minimum/maximum transaction limit

The minimum transaction limit is used to set the minimum amount a user can purchase. If you set it to 100 and you have set the currency to CNY, the minimum transaction amount that the user can initiate a transaction with you is 100 CNY.

The maximum transaction limit is used to set the maximum transaction amount you want to accept. Please note that if the amount you enter is greater than the number of Bitcoins in your wallet, the actual maximum amount will be set to the balance in your wallet.


Terms of the transaction

The text the buyer saw before initiating the transaction with you. It is recommended that you write a description for the buyer so that the buyer knows how to trade and any specific instructions from you.

For example, you will only be able to release Bitcoin after you have submitted your receipt as proof of payment, or if you need a buyer to provide proof of identity, you can specify in the terms of the transaction. You can browse the ads of other traders to understand the payment methods you want to use in order to clearly understand the content of the terms of the transaction.


Payment details

Enter specific information here related to how the buyer should pay, which can be your bank account number or other payment information.


Expiration of pay

How long the buyer can complete the payment before the transaction is automatically cancelled. Set the time yourself.


Enable security options

By checking this check box, you can set limits for ads, only for trusted traders, trader certification levels, and meet the limits you set before users can initiate transactions.


Minimum transaction volume

Ask the buyer how much trading volume was done in BTC before. Setting this value to be greater than 0 will filter out all first-time buyers with zero volume.


The lowest evaluation score

Percentage of buyers demanding buyers (0-100). This item is not used for first time buyers who have not yet scored.


New buyer limit

If the trader has not made a transaction before, this value will limit his first transaction volume. Set a lower value to limit the risk of fraud.
